十一、字符串操作
#字符串操作
1.去除空格(strip、lstrip、rstrip)
str.strip():去除字符串两端的空格
str.lstrip():去除字符串左端空格
str.rstrip():去除字符串右端空格
2.字符串链接(+)
str = str1 + str2
3.字符串查找(index)
str = ' a b c d '
pos = str.index("b")#类似于list查找，用index方法，返回值为所在位置
print(pos)#3
4.字符串长度(len)
len = len(str)
5.字符串转换成全大写(upper)
str.upper()
6.字符串转出全小写(lower)
str.lower()
7.字符串首字母大写(capitalize)
str.capitalize()
8.字符串拆分(split)
str2.split(',') #按照字符串中的",",将字符串拆分成n个元素的list返回。
9.字符串链接(join)
'-'.join(list)#将list的字符元素以“-”链接起来 
10.类型转换(强转+指定进制)
#强制转换
str3 = '123'
int(str3)#字符串强转int
str(3)#数字强转字符串
#指定转换进制
int("1234",16)
int("101010",2)
int("101010",8)
11.格式化字符串（占位符）
str = 'Hello %s' % "world"

#判断字符串性质
1.判断字符串的起始字符
str.startswith('a')#判断字符串是否以"a"开始，区分大小写
2.判断字符串的结束字符
str.endswith('E')#判断字符串是否以"E"结束，区分大小写
3.判断字符串是否全是数字或字符
str.isalnum()
4.判断字符串是否全字符
str.isalpha()
5.判断字符串是否全数字
str.isdigit()
6.判断字符串是否全空白
str.isspace()
7.判断字符串是否全小写
str.islower()
8.判断字符串是否全大写
str.isupper()
9.判断字符串是否只有首字符大写
str.istitle()

#字符串数子之间的转换函数
1.int(str,[base])#将字字符串str按照base进制转换成int类型
2.long(str.[base])#将字字符串str按照base进制转换成long类型 
3.float(str)#将字符串str转换成float类型
4.complex(real,imag)#构造复数
5.str(int)#将int类型数据转换成字符串
6.eval('123+1234')#1357，字符串中有效的表达式可以计算
7.repr('123+1234')#123+1234，字符串中的表达式作为字符串处理  
8.tuple(s)#将序列 s 转换为一个元组    
9.list(s)#将序列 s 转换为一个列表    
10.chr(x)# 将数值转换成Ascii码
11.unichr(x)#将一个整数转换为Unicode字符    
12.ord(x)#将Ascii码转换成数值  
13.hex(x)#将一个整数转换为一个十六进制字符串    
14.oct(x)# 将一个整数转换为一个八进制字符串 

作业：
字符串按单词反转（必须保留所有空格）。'I love china!'转化为'china! love I'.
str = 'I love china!'
ls = str.split(' ')#字符串拆分
ls.reverse()#list翻转
str = ' '.join(ls)#字符串链接
